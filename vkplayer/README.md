# VK Player Plugin for Lampa

Плагин для просмотра видео из ВКонтакте в медиаплеере Lampa.

## Возможности

- Авторизация через VK OAuth
- Поиск видео
- Просмотр личных видео
- Просмотр видео из сообществ
- Просмотр популярных видео
- Выбор качества (240p - 1080p, HLS)
- Интеграция с Lampa Player

## Установка

1. Скопируйте файл `vkplayer.js` на хостинг
2. В Lampa откройте: **Настройки → Расширения → Добавить плагин**
3. Вставьте URL к `vkplayer.js`
4. Подтвердите установку

## Настройка VK API

1. Перейдите в [VK API](https://vk.com/dev)
2. Создайте Standalone приложение
3. Скопируйте `App ID`
4. Вставьте его в плагин (переменная `VK_APP_ID`)

## Авторизация

1. Откройте настройки плагина
2. Нажмите "Войти через VK"
3. Перейдите по ссылке и разрешите доступ
4. Скопируйте `access_token` и `user_id` из URL
5. Вставьте их в форму

## Использование

- **Поиск**: В главном меню выберите "VK Поиск"
- **Мои видео**: В настройках плагина
- **Популярное**: Доступно через поиск

## Требования

- Lampa v3.0+
- Подключение к интернету
- Аккаунт VK

## Ограничения

- Некоторые видео могут быть недоступны из-за ограничений VK API
- DRM-контент не поддерживается
- Лайв-стримы работают через HLS
